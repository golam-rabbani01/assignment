DEVCONF 2026 is a conference website created using HTML and CSS. The project includes a navigation bar, hero section, speakers section, and pricing section. Users can view conference information, explore speakers, and choose different ticket plans. The website is designed with a clean, modern, and responsive layout using Flexbox and CSS styling.
